## Introduction

An R Package to implement ls() style regression.

## Installation

```r
devtools::install_github("shaiq681/Regression")
```

## Usage

- For creating a new Regression Object

```r
linreg_obj <- linreg$new(formula= Sepal.Length~Petal.Width, data = iris, use.qr = F)
```
## LiU Themes
